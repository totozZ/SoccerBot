# SoccerBot
本项目基于 FRC 控制系统与 Unity VR 技术，使用 C++ 开发足球机器人控制程序，通过 NetworkTables 实现 RoboRIO 与 Unity 间的数据通信，并结合 Meta Quest 2 构建机器人运动与足球发射的实时 VR 可视化场景。
本项目旨在设计并实现一套基于 FRC 控制架构的足球机器人系统，采用 RoboRIO 2.0 作为机器人主控制器，并使用 Falcon 500 电机完成机器人运动及足球发射功能。

系统结合 Meta Quest 2 与 Unity 构建虚拟现实场景，实现机器人状态与足球发射过程的实时可视化。用户佩戴 VR 设备后，可在虚拟场景中观察足球机器人的运动状态、足球飞行轨迹以及模拟比赛环境，从而形成“现实机器人 + 虚拟场景”的交互展示效果。

机器人控制部分基于 FRC 官方开发框架 WPILib，使用 C++ 进行程序开发；虚拟场景部分基于 Unity 引擎开发，并通过 FRC 常用的 NetworkTables 通信机制实现机器人数据与 Unity 场景之间的数据同步。项目中同时结合 Limelight 3A 视觉设备，用于目标检测及机器人状态数据获取。

本项目重点实现以下功能：

足球机器人基础运动控制
足球发射机构控制
Unity 虚拟场景构建
VR 场景实时观察
足球轨迹可视化
基于 NetworkTables 的实时通信同步
